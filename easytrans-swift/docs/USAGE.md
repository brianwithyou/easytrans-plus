## 安装与首次启动

1. 将 `EasyTrans.app` 放入「应用程序」文件夹，或从 Xcode 直接运行
2. 启动后，菜单栏会出现气泡图标（`character.bubble`），主窗口同时打开
3. 打开 **EasyTrans → Settings…**（`⌘ + ,`），填入 DashScope API Key
4. 若需使用全局快捷键（`⌘⇧D`、`⌘⇧V`），按下方 [辅助功能授权](#辅助功能授权) 完成授权

> EasyTrans 默认不在 Dock 显示图标，关闭主窗口后应用仍在菜单栏运行。

## 主窗口翻译

主窗口采用左右分栏布局：

| 区域 | 说明 |
|------|------|
| 左侧「原文」 | 输入待翻译文本 |
| 右侧「译文」 | 流式显示翻译结果，支持 Markdown 加粗渲染 |
| 语言栏 | 源语言固定为英语；目标语言可在设置中选择 |
| 底部 | 「翻译」按钮，或按 `⌘ + Enter` |

### 操作步骤

1. 在左侧输入或粘贴文本
2. 在语言栏选择目标语言（默认中文）
3. 点击「翻译」或按 `⌘ + Enter`
4. 译文实时流式显示；点击译文区右上角复制图标可复制纯文本

### 清空

点击语言栏右侧「清空」可清除原文与译文，并取消进行中的翻译任务。

## 全局快捷翻译

快捷键：**`⌘ + Shift + D`**

在任意应用中选中文字后按下快捷键，EasyTrans 会：

1. 读取当前选中的文本
2. 自动判断翻译方向（见 [语言与翻译模式](#语言与翻译模式)）
3. 调用 Qwen 模型翻译
4. 将译文写入系统剪贴板
5. 打开主窗口并展示翻译过程与结果

### 使用场景

| 场景 | 行为 |
|------|------|
| 浏览器 / 编辑器中选中英文 | 翻译为目标语言，单词附带音标 |
| 选中中文 | 翻译为英语 |
| EasyTrans 主窗口内选中文字 | 无需辅助功能权限，直接翻译 |

### 触发方式

- 全局快捷键 `⌘⇧D`
- 菜单栏 →「翻译选中文字 ⌘⇧D」
- 菜单 **翻译 → 翻译选中文字**

### 辅助功能授权

在其他应用中读取选中文字需要 **辅助功能** 权限：

1. 打开 **EasyTrans → Settings…**
2. 在「快捷翻译」区域点击「打开系统设置」
3. 将 EasyTrans 加入辅助功能列表并开启
4. **完全退出** EasyTrans（`⌘Q`）后重新打开

> **开发提示**：从 Xcode `⌘R` 构建时，应用输出到项目 `build/Debug/` 目录。对该路径授权一次后，日常重新编译通常无需重复授权。若从 DerivedData 运行，每次编译路径可能变化，授权容易失效。

## 剪贴板历史

快捷键：**`⌘ + Shift + V`**

EasyTrans 在后台监听系统剪贴板，自动记录复制过的纯文本，最多保留 **5000** 条。

### 打开历史面板

- 按 `⌘⇧V`（再次按下可关闭）
- 菜单栏 →「剪贴板历史 ⌘⇧V」
- 菜单 **剪贴板 → 查看剪贴板历史**

面板在鼠标附近弹出，支持搜索过滤。

### 粘贴历史条目

1. 在目标应用的输入框中定位光标
2. 按 `⌘⇧V` 打开历史面板
3. **双击** 某条记录，内容将粘贴到打开面板时锁定的应用

| 操作 | 说明 |
|------|------|
| 单击 | 选中条目 |
| 双击 | 粘贴到原应用 |
| `Esc` | 关闭面板 |
| 点击面板外 | 关闭面板 |
| 切换到其他应用 | 自动关闭面板 |

### 数据存储

历史记录持久化至：

```
~/Library/Application Support/com.easytrans.app/clipboard-history.json
```

应用退出时自动落盘。从剪贴板历史粘贴的内容不会再次被记入历史。

## 菜单栏

点击菜单栏图标可访问：

| 菜单项 | 功能 |
|--------|------|
| 翻译选中文字 ⌘⇧D | 触发全局快捷翻译 |
| 剪贴板历史 ⌘⇧V | 打开剪贴板历史面板 |
| 打开主窗口 | 显示翻译主界面 |
| 快捷键状态 | 显示 Carbon / EventTap / Global / Local 注册情况 |
| 打开辅助功能设置… | 未授权时显示，跳转系统设置 |
| 刷新快捷键 | 重新注册全局监听 |
| 退出 EasyTrans | 退出应用 |

## 设置页

通过 **EasyTrans → Settings…**（`⌘ + ,`）打开。

### 通用

- **登录时自动启动**：开启后应用随 macOS 登录启动，可在「系统设置 → 通用 → 登录项」中管理

### DashScope 配置

| 字段 | 说明 |
|------|------|
| API Key | DashScope 密钥，必填 |
| 模型 | `qwen-turbo`（快）/ `qwen-plus`（均衡）/ `qwen-max`（强）/ `qwen-long`（长文本） |
| API Base URL | OpenAI 兼容接口地址，国内与国际节点见 README |

### 快捷翻译

显示全局快捷键、辅助功能授权状态，并提供：

- **在 Finder 中显示应用** — 便于拖入辅助功能列表
- **请求授权** — 触发系统授权弹窗
- **打开系统设置** — 跳转辅助功能页面
- **刷新状态** — 重新检测授权并刷新快捷键

## 语言与翻译模式

### 自动语言方向

EasyTrans 根据输入内容自动决定翻译方向：

| 输入语言 | 翻译方向 |
|----------|----------|
| 与目标语言相同（如目标为中文，输入中文） | → 英语 |
| 英语 | → 目标语言 |
| 其他语言 | 自动检测 → 目标语言 |

### 单词增强模式

当输入被判定为「单词或短语」时，输出格式会增强：

**英文 → 目标语言**（如中文）：

```
**apple** /ˈæp.əl/
n. 苹果
```

**目标语言 → 英文**（多义项）：

```
bank n. /bæŋk/
shore n. /ʃɔːr/
```

**句子或段落**：仅输出译文，不附带音标。

判定规则概要：含换行、句末标点、超过 5 个词、或 CJK 字符超过 12 个等，视为句子/段落。

### 支持的目标语言

中文、日语、韩语、法语、德语、西班牙语、俄语、阿拉伯语、越南语。

## 常见问题

### 提示「未配置 API Key」

在设置页填入有效的 DashScope API Key。可在 [DashScope 控制台](https://dashscope.console.aliyun.com/apiKey) 创建。

### 全局快捷键无响应

1. 确认已授予辅助功能权限
2. 完全退出并重新打开 EasyTrans
3. 菜单栏 →「刷新快捷键」
4. 查看菜单栏「快捷键状态」是否至少有一项显示 ✓

### 辅助功能已开启但仍显示未授权

1. 在辅助功能列表中**删除**旧的 EasyTrans 条目
2. 设置页点击「在 Finder 中显示应用」
3. 将新的 `EasyTrans.app` 拖入辅助功能列表
4. `⌘Q` 退出后重新启动

### 翻译失败 / 网络错误

- 检查 API Key 是否有效、账户余额是否充足
- 确认 Base URL 与所在区域匹配（国内 / 国际）
- 检查网络是否能访问 `dashscope.aliyuncs.com`

### 剪贴板历史无法粘贴到原应用

1. 确保打开历史面板**之前**已在目标输入框定位光标
2. 确认已授予辅助功能权限
3. 部分应用（如终端、受保护输入框）可能不支持模拟粘贴

### 关闭主窗口后应用还在运行？

这是预期行为。EasyTrans 作为菜单栏应用常驻后台。要完全退出，请使用菜单栏「退出 EasyTrans」或 `⌘Q`。

### 如何卸载

1. 退出 EasyTrans
2. 删除 `/Applications/EasyTrans.app`（或你的安装位置）
3. 可选：删除 `~/Library/Application Support/com.easytrans.app/` 清除本地数据
4. 在「系统设置 → 通用 → 登录项」中移除开机启动项
